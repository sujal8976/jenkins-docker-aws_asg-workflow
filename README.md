# URL Shortener (TypeScript)

This repository contains a simple URL shortener with a TypeScript Express backend and a React + Vite frontend.

Quick start (run backend and frontend separately):

1. Backend

```bash
cd backend
npm install
npm run dev
```

2. Frontend

```bash
cd frontend
npm install
npm run dev
```

Run tests:

```bash
cd backend
npm test

cd ../frontend
npm test
```
